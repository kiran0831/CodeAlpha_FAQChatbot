# 🤖 FAQ Chatbot
### CodeAlpha AI Internship — Task 2

A desktop FAQ Chatbot built with Python, Tkinter, and NLTK that uses Cosine Similarity to match user questions with the most relevant answer.

## Features
- NLP preprocessing with NLTK (tokenization, stemming, stopword removal)
- Cosine Similarity based intent matching
- 20 built-in CodeAlpha FAQs
- Quick suggestion buttons
- Clean dark chat UI with timestamps

## Tech Stack
- Python 3.8+
- Tkinter — GUI
- NLTK — NLP preprocessing
- Cosine Similarity — answer matching

## How to Run
pip install -r requirements.txt
python main.py

## How It Works
1. User types a question
2. NLTK tokenizes and stems the input
3. Cosine similarity matches it against all FAQ questions
4. The best matching answer is displayed
